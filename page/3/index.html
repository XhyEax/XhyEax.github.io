使用__attribute__配置结构体，禁用编译器自动对齐

发表于 2020-07-21

概述

结构体声明中加上__attribute__((packed))，可以防止编译器对结构进行字节对齐优化。
使用__attribute__((aligned(8)))(8为对齐字节数)，可以强制编译器按指定字节对齐。

结构体定义

使用 packed

使用sizeof获取结构体的大小，返回7

不使用 packed

根据结构体中最长的类型(int)，进行对齐，获取到的结构体大小为8

指定对齐大小

不使用指定对齐字节时，结构体大小为3*4=12，指定后变为16

win10 切换诊断数据为完整

发表于 2020-04-15

概述

安装windows预览版，需要切换诊断数据为完整，而本人切换到完整后又会自动跳回基本。搜索后找到解决方案。
原因是Connected User Experiences and Telemetry服务被禁用

解决步骤

Win+R,services.msc,找到Connected User Experiences and Telemetry，设置为自动(延迟启动)，再点击启动此服务即可成功切换
