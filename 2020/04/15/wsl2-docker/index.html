启动docker并配置权限

添加当前用户进docker组

  1. 查看是否创建docker组
    cat /etc/group | grep docker
  2. 创建docker组
    sudo groupadd docker
  3. 将用户添加到docker分组
    sudo usermod -aG docker 用户名
  4. 查看是否生效
    cat /etc/group | grep docker
  5. 重启docker
    sudo systemctl restart docker
  6. 给docker.sock添加权限
    sudo chmod a+rw /var/run/docker.sock
