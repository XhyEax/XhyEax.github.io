npm package.json scripts 传递参数

发表于 2021-11-21 阅读次数：

概述

直接加参数运行（如：npm run view test），实际上是拼接字符串到末尾，不够灵活。

使用npm_config环境变量传递参数，支持多个参数、多条命令顺序执行。
